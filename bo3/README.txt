### type python main.py in terminal to run the game

### Left and right arrow to control player

### Player has 3 lives to complete 5 predesigned levels
### Each level ends when all bricks are destroyed.
### Player lose 1 life if the ball drops out of the screen
### The game ends if all 3 lives are lost

### If the player lose all lives and the game ends, 
### the player can enter his/her name and the name along
### with the score will be saved to a .txt file
